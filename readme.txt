== Installation ==
1. Download the plugin as a .zip-file. Navigate to your plugins in Wordpress. You can upload the plugin here in the .zip-file, or use FTP to upload it manually to your plugins folder
2. You can now activate the plugin in the plugins-dashboardDanach aktivieren Sie das Plugin in der Plugin-Übersicht
3. Navigate to WP etracker in the Wordpress sidebar
4. Accept the AGBs (allgemeinen Geschäftsbedingungen) to activate the tracking of the plugin
5. Enter your etracker.com account key and click speichern (save). Now your plugin is setup and you are ready to go.
